## 1. 环境安装

### 1.1 下载安装conda
[清华anacodna开源站](https://mirror.tuna.tsinghua.edu.cn/help/anaconda/)

### 1.2 配置开发环境
安装好anaconda之后，使用以下命令创建python 虚拟环境：
```bash
conda create -n quant python=3.12 -y

conda activate quant

pip install -r requirements.txt
```
